Bank management system is a digital banking application developed using java sql and jdbc technologies.
this allows users to manage their bank accounts digitally and make transactions efficiently.
users can perform all operations that can be performed in a physical bank.
I have used JDBC framework to connect the java application with database and update and retrieve realtime data

AWT
